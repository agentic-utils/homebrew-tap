# Homebrew Tap

Homebrew tap for agentic-cli.

## Installation

```bash
brew tap agentic-cli/tap
```

## Packages

| Formula | Description |
|---------|-------------|
| [ccs](https://github.com/agentic-cli/ccs) | Search and resume Claude Code conversations using fzf |

## Usage

```bash
brew install agentic-cli/tap/ccs
```
