# Homebrew Tap

Homebrew tap for agentic-utils.

## Installation

```bash
brew tap agentic-utils/tap
```

## Packages

| Formula | Description |
|---------|-------------|
| [ccs](https://github.com/agentic-utils/ccs) | Search and resume Claude Code conversations using fzf |

## Usage

```bash
brew install agentic-utils/tap/ccs
```
