# Homebrew Tap

Personal Homebrew tap for brtkwr.

## Installation

```bash
brew tap brtkwr/tap
```

## Packages

| Formula | Description |
|---------|-------------|
| [ccs](https://github.com/brtkwr/ccs) | Search and resume Claude Code conversations using fzf |

## Usage

```bash
brew install brtkwr/tap/ccs
```
