# Homebrew Tap

Homebrew tap for agents-cli.

## Installation

```bash
brew tap agents-cli/tap
```

## Packages

| Formula | Description |
|---------|-------------|
| [ccs](https://github.com/agents-cli/ccs) | Search and resume Claude Code conversations using fzf |

## Usage

```bash
brew install agents-cli/tap/ccs
```
